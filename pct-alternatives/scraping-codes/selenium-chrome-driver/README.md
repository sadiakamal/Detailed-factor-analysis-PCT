### installation

We need `selenium` and `chrome-driver` if not installed already.

- `pip install selenium`
-  install chrome-driver from here: https://googlechromelabs.github.io/chrome-for-testing/#stable
-  [modified selenium test code](test-chrome-driver.py) from here: https://sites.google.com/chromium.org/driver/getting-started?authuser=0
